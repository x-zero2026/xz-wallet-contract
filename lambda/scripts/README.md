# Lambda Scripts

This directory contains test and utility scripts for the XZ Wallet Lambda functions.

## Files

### SQL Scripts
- `check_balance.sql` - Query to check user balance
- `check_task.sql` - Query to check task details

### JavaScript Scripts
- `check-balance.js` - Script to check blockchain balance
- `check-user-address.js` - Script to check user Ethereum address

### Go Scripts
- `test-token.go` - Script to test JWT token generation and validation

## Usage

These scripts are for testing and debugging purposes only. They are not part of the production deployment.
